# linguistics

本リポジトリは、東京工業大学文系教養科目「言語学B」におけるポスター発表「カラオケで人気な曲の歌詞の表現特性」のために作成したプログラムである。

## environments

- macOS Mojave 10.14.5
- Python 3.7.4
- MeCab 0.996

## init

```sh
$ brew install anyenv
$ exec $SHELL -l
$ anyenv install pyenv
$ exec $SHELL -l
$ pyenv install 3.7.4
$ brew install mecab mecab-ipadic
```
